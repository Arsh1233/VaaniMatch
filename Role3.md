### Role 3: Frontend / UI/UX Engineer (Bright & Exclusive UI)
*Focus: Glass-morphism, neon gradients, and micro-interactions.*

**Prompt 3.1 (Global Bright Theme & Glass-morphism):**
> "Generate the global `tailwind.config.js` and a base `globals.css` for a bright, exclusive recruiter dashboard. Define a custom gradient palette: Electric Violet (#8B5CF6) to Coral Pink (#FB7185) to Bright Cyan (#22D3EE). Implement a glass-morphism utility class with `backdrop-filter: blur(16px)`, a white/transparent background, and a 1px solid white/opacity-20 border. Set the background to an animated luminous gradient mesh."

**Prompt 3.2 (JD Input Component with Voice Toggle):**
> "Write a React + TypeScript component for the JD input form. Features: (1) Drag-and-drop PDF upload area that extracts text, (2) A bright, pulsating voice-input button that triggers the Web Audio API, (3) A language toggle switcher (English/हिन्दी) with sliding animations, and (4) A 'Magic Enrich' button that expands the JD via the LLM API. Style it with glowing focus rings and neon accents."

**Prompt 3.3 (Ranked Candidate Cards with Radial Scores):**
> "Build a dynamic candidate card component in React using `recharts` or `visx`. Each card displays the candidate name, verified badge, and 5 radial progress bars representing Semantic, Trajectory, Behavior, Graph, and Verification scores. Include an expanding accordion titled 'Explain Ranking' that reveals the counterfactual text and a slider to adjust 'Desired Seniority' which dynamically updates the ranking in real-time."

**Prompt 3.4 (Candidate Voice Onboarding Mobile UI):**
> "Design a mobile-first, bright landing page for candidate sign-up. Use a bold gradient background. Center a large, glowing, animated microphone button with a ripple effect. Underneath, show a conversational chat bubble that updates as the AI extracts info (e.g., 'Got it! You are a Software Engineer...'). Display an exclusive 'e-Shram Verified' badge with a glittering gold animation once the Aadhaar OTP is linked."
